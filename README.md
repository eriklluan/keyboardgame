# KeyGame - Treino de Atalhos de Teclado

Um jogo divertido e interativo para aprender os atalhos de teclado mais essenciais do computador.

## Funcionalidades

- **Dois modos de jogo**: Atalho Rápido e Quiz
- **Atalhos incluídos**:
  - Ctrl + C (Copiar)
  - Ctrl + V (Colar)
  - Ctrl + Z (Desfazer)
  - Ctrl + S (Salvar)
  - Ctrl + A (Selecionar Tudo)
  - Ctrl + X (Recortar)
  - Ctrl + P (Imprimir)
  - Ctrl + T (Nova Aba)
  - Ctrl + W (Fechar Aba)
  - Alt + F4 (Fechar Janela)
  - Ctrl + B (Negrito)
  - Ctrl + U (Sublinhar)
  - Ctrl + N (Novo)
  - Ctrl + O (Abrir)
  - Ctrl + F (Buscar)
  - Ctrl + R (Recarregar)
  - Ctrl + Tab (Próxima Aba)
  - Alt + Tab (Alternar Janelas)
  - F5 (Recarregar)
  - F11 (Tela Cheia)
  - Ctrl + Shift + T (Reabrir Aba)
  - Ctrl + D (Favoritos)
  - Ctrl + Shift + I (Inspecionar)
  - Ctrl + L (Barra de Endereço)

## Como Jogar

1. Escolha o modo de jogo
2. No modo "Atalho Rápido", veja a ação e pressione o atalho correto
3. No modo "Quiz", veja o atalho e escolha a ação correspondente
4. Ganhe pontos e combos por acertos consecutivos
5. Tenha cuidado com o tempo e as vidas!

## Deploy na Vercel

Este projeto é um site estático e pode ser facilmente implantado na Vercel.

1. Faça upload do código para um repositório Git (GitHub, GitLab, etc.)
2. Conecte o repositório à Vercel
3. A Vercel detectará automaticamente o `index.html` e fará o deploy

## Desenvolvimento Local

Para rodar localmente:

```bash
npm install
npm run dev
```

Ou simplesmente abra o `index.html` no navegador.
