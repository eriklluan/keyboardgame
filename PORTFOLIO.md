# KeyGame - Portfolio Case Study

## Resumo

O **KeyGame** e um aplicativo web gamificado criado para ajudar estudantes iniciantes a desenvolver fluencia em atalhos de teclado. Ele nasceu de uma necessidade real observada no projeto de extensao **Programe seu Futuro (Unijui)**, durante atividades em que estudantes estavam aprendendo programacao e usando ferramentas como o MIT App Inventor.

**Demo:** https://keyboardgame-main.vercel.app

**Repositorio:** https://github.com/eriklluan/keyboardgame

## Problema

Durante as aulas, muitos alunos tinham dificuldade com atalhos basicos como copiar, colar, desfazer e salvar. Isso reduzia a velocidade das atividades, quebrava o ritmo da aula e deixava tarefas simples mais cansativas.

## Solucao

O KeyGame transforma o treino de atalhos em uma experiencia interativa com:

- modo pratica sem login;
- entrada por turma com codigo;
- painel do professor;
- ranking global e por turma;
- quiz, desafio rapido e aprendizado guiado;
- progressao por dificuldade;
- sistema de pontuacao com combo, bonus e penalidade leve;
- interface em portugues e ingles.

## Meu Papel

Atuei em todo o ciclo do projeto:

- observacao do problema em sala de aula;
- definicao dos modos de jogo e dos fluxos de aluno/professor;
- implementacao da interface em HTML, CSS e JavaScript;
- criacao das APIs serverless na Vercel;
- integracao com Supabase/PostgreSQL;
- configuracao de deploy;
- escrita de testes automatizados;
- documentacao do projeto para GitHub e portfolio.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Node.js
- Vercel Serverless Functions
- Supabase/PostgreSQL

## Destaques Tecnicos

- Aplicacao single-page sem framework.
- Backend serverless separado por dominio: jogador, ranking e professor.
- Variaveis de ambiente para proteger credenciais e chave de professor.
- Ranking persistido em banco de dados.
- Testes para entrada de jogador, pontuacao, rankings, fluxos de professor e elementos principais da interface.
- Deploy publico com Vercel.

## Impacto

O projeto foi aplicado com **22 estudantes** em um contexto educacional real. Ele mostra como uma dificuldade simples, mas recorrente, pode virar uma ferramenta completa de aprendizagem quando o desenvolvimento parte da observacao do usuario.

**KeyGame** is a gamified web app designed to help beginner students improve keyboard shortcut fluency. It was created from a real classroom need in the Programe seu Futuro extension project and includes practice mode, classroom access, teacher tools, rankings, guided learning, bilingual UI, Supabase persistence, serverless APIs, automated tests, and production deployment on Vercel.
